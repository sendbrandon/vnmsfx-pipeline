The System

The videos are the output.
The system is the product.

Generative models drift, forget, and ignore instructions. The studio wraps them in an operating system — plain-text law, versioned in git, that an AI agent boots from on every session. Four layers:

01 · THE LAWA constitution that auto-loadsStage discipline, quality gates, cost controls — and drop-detection: hand the system an image, a take, or a trigger word, and the correct stage fires with zero instructions.
02 · THE LINESeven stages, locked gatesLook development → cast → series bible → episode prompt → generation → screening → ship. No stage starts before the previous gate locks. Nothing runs from memory — every stage executes from a versioned methodology file.
03 · THE SCREENEvery output is screened — wins includedFrame sheets, transcripts, and kill criteria decided before generating, not after. Verdicts and failure tags are logged, one line per take, forever.
04 · THE MEMORYFailures become lawEvery screening writes its lesson to a ledger the system re-reads before all future work. A scheduled review counts failure patterns and promotes anything seen twice into permanent law.
THE RATCHETQuality mechanically cannot slip backward. The top historical failure class was eliminated by a single written law — zero recurrences since.
State on disk, never in the model

The agent is disposable; the system is not. Every project boots from files, so it survives session resets, context loss, and model swaps.

Deterministic gates around a stochastic core

Scripts measure what judgment used to eyeball — prompt caps counted, screenings auto-extracted, finishing grades triggered by a single word.

An eval layer that coaches its operator

The scoreboard doesn't just track the AI's failures — it surfaces the human's recurring blind spots and says them out loud.

Human-in-the-loop by design

The system pitches slates of options with a recommendation. Money is never spent and nothing publishes without an explicit human pick.

Taste, converted into process

Directorial method — coverage, rhythm, signal theory — encoded as advisory checkpoints that run on every episode automatically.

Replicable in any domain

The whole studio rebuilds itself from a single bootstrap: a structured intake interview whose answers fill verbatim templates, verified by checklist. Video was the test case.
